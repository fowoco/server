# 지원과 질문

로컬 실행·환경설정·API 연동 질문은 [Q&A Discussion](https://github.com/fowoco/server/discussions/19)에 남겨 주세요.

질문에는 다음을 포함하면 답변하기 쉽습니다.

- 하려던 일과 사용한 명령 또는 API
- 예상 결과와 실제 결과
- OS, Java 버전, 활성 Spring Profile
- `request_id`
- 이미 시도한 방법
- 개인정보와 Secret을 제거한 오류 일부

재현 가능한 제품 오류는 Bug Issue Form을 사용합니다. 보안 취약점이나 개인정보 노출 가능성은 공개 Issue나 Discussion에 작성하지 말고 [보안 정책](SECURITY.md)을 따릅니다.
